Read First

The PieBar_III.zip file contains the following files:

  1. CompileRun_piebar.bat - a batch file that when double clicked will compile and execute the Piechart II program.
  2. PieChart_III_Help.html - this file provides help on how to get started.
  3. PieBarGraph.java - the Java source file for the Piechart II program.
  4. PieBar.html - the .html file that runs the Piechart II applet.
  5. run_Javadoc.bat - the batch file that executes the Javadoc file.
  6. View_PieBar_Applet - click on this .HTML file to view the PieBarGraph application.
    1. First, double click on the CompileRun_piebar.bat file.
    2. Then, double click on View_PieBar_Applet file (after a successful compile).

How to compile and run

There are two ways to run this application:

  1. You can double-click on the CompileRun_piebar.bat file that is supplied with the PieChart_II.zip file.
  2. You can load the PieBarGraph.java file into an IDE and run the program from that environment.

If you do not have the Java documentation

The Help and Tutorial is not intended to repeat the documentation found in:

  1. J2SE 1.3.1
  2. JavaTM 2 Platform, Standard Edition, v 1.4.2 API Specification

A Basic Set of Test Cases

In this section, I have included a basic set of test cases for Pie/Bar Chart III. The following are the test cases:

  1. Test for all of the text fields being null/blank/empty.
    • Leave all of the fields blank.
  2. Test for one of the text fields being null/blank/empty.
    • Enter whole numbers: 45, 95 145, 195 and blank.
  3. Test when all of the text entry fields contain whole numbers.
    • Enter whole numbers: 45, 95 145, 195, and 255.
    • Press the Create Pie Chart button.
  4. Test for one of the text fields not being a whole number.
    • Enter whole numbers: 45, 95 145, 19g and 255.
    • Press the Create Pie Chart button. A pie chart should appear.
    • Press the Create Bar Chart button. A bar chart should appear.
  5. Test the contents being printed.
    • Enter whole numbers: 45, 95 145, 195, and 255.
    • Press the Create Pie Chart button. A pie chart should appear.
    • Press the Print button. The pie chart should be printed.
    • Press the Create Bar Chart button.
    • Press the Print button. The bar chart should be printed.
  6. Test for Help being displayed.
    • Enter whole numbers: 25, 55 100, 175, and 250.
    • Press the Help button. The Internet Explorer should display this help file.
  7. Test the About button.
    • Enter whole numbers: 25, 55 100, 175, and 250.
    • Press the About button. The Internet Explorer should try to connect to my site.
  8. Test the Clear button.
    • Enter whole numbers: 45, 95 145, 195, and 255.
    • Press the Create Pie Chart button.
    • Press the Clear button. The text fields and the pie chart should be cleared from the screen.
  9. Test the Exit button.
    • The Pie/Bar application should shut down.
